Download primesieve

The latest primesieve version is 5.4.2 released on the 4th April 2015, see the ChangeLog for what's new. For each supported operating system there are two primesieve binaries available, a GUI version and a console version. Have a look at the screenshots page for pictures. Please report any bugs to the author.

Old primesieve releases can be downloaded from http://dl.bintray.com/kimwalisch/primesieve
